# ⚡ Быстрый старт Zapretgram

## 🎯 За 2 минуты до чата:

### Шаг 1️⃣: Первая регистрация (браузер 1)
```
1. Открыть index.html
2. Нажать "Создать аккаунт"
3. Никнейм: Маша
4. Пароль: 1234
5. Подтвердить пароль: 1234
6. Нажать "Зарегистрироваться"
```

### Шаг 2️⃣: Вход
```
1. Никнейм: Маша
2. Пароль: 1234
3. Нажать "Войти"
```

### Шаг 3️⃣: Вторая регистрация (браузер 2 или приватное окно)
```
Повторить тоже самое, но с никнеймом "Говно" и паролем "5678"
```

### Шаг 4️⃣: Отправка сообщения (браузер 1 = Маша)
```
1. В поле поиска слева написать "Говно"
2. Нажать на "➕ Говно" в разделе "🔍 Найти пользователей"
3. Написать сообщение: "Привет, Говно!"
4. Нажать ➤ (отправить)
```

### Шаг 5️⃣: Получить ответ (браузер 2 = Говно)
```
1. Обновить страницу F5
2. Вводите пароль 5678
3. В левой панели появится "Маша"
4. Нажать на "Маша"
5. Увидите сообщение "Привет, Говно!"
6. Ответить: "Привет, Маша!"
```

---

## 🔑 Важные горячие клавиши:

| Клавиша | Действие |
|---------|----------|
| `Enter` | Отправить сообщение |
| `F5` | Обновить страницу (для синхронизации) |
| `Правый клик` | Удалить контакт из списка |

---

## 🔍 Поиск и добавление контактов:

1. **Найти пользователя**: напишите в поиске его никнейм
2. **Типы результатов**:
   - 👥 Мои контакты - текущие чаты
   - 🔍 Найти пользователей - доступные для добавления
   - 💬 Сообщения - содержащие текст

---

## 📌 Особенности:

✅ **Чаты сохраняются** - закройте браузер, откройте снова - всё там  
✅ **История сообщений** - все сообщения хранятся в браузере  
✅ **Синхронизация** - если открыть две вкладки одного браузера - будет синхронизация  
⚠️ **Между браузерами** - нужно обновлять F5 для синхронизации через localStorage  

---

## 🐛 Если не работает:

1. **"Пользователь не найден"** - зарегистрируйте его в другом браузере/окне сначала
2. **Контакт не появляется** - обновите страницу F5
3. **Нет синхронизации** - убедитесь, что используете localStorage (все браузеры это поддерживают)

---

## 🚀 Для кросс-браузерной синхронизации в реальном времени:

Запустите WebSocket сервер:
```bash
node server.js
```

Тогда сообщения будут синхронизироваться сразу, без F5!
