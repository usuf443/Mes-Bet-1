# 🎉 Новые функции Zapretgram v1.1

## ✅ Что было добавлено/исправлено:

### 1. **Пароли при регистрации** 🔐
- Вернул систему пароля в регистрацию
- При входе нужно вводить никнейм **и** пароль
- Пароли хранятся в localStorage (в реальном приложении нужно хешировать)

### 2. **Кросс-браузерные чаты** 💬
- Теперь можно открыть приложение в **двух разных браузерах** и создать два разных аккаунта
- Например: браузер 1 = "Маша", браузер 2 = "Говно"
- Маша может найти Говно в поиске и написать ему
- Говно ответит Маше, и сообщения будут видны обоим
- **Синхронизация работает через:**
  - localStorage (сохранение сообщений для каждого пользователя)
  - BroadcastChannel (синхронизация между вкладками одного браузера)
  - WebSocket (синхронизация между разными браузерами, если сервер работает)

### 3. **Сохранение чатов при долгом отсутствии** 💾
- Чаты больше **не пропадают** даже если долго не заходил
- История сообщений сохраняется в localStorage
- При входе все контакты и сообщения восстанавливаются автоматически

### 4. **Поиск и добавление контактов** 🔍
- **Поле поиска** в левой панели теперь:
  - Показывает **мои контакты** (текущие чаты)
  - Показывает **все зарегистрированные пользователи** для добавления новых контактов
  - Показывает **все сообщения**, содержащие поисковый запрос

### 5. **Улучшенный интерфейс контактов** ✨
- **Зеленая точка** рядом с аватаром = пользователь онлайн
- **Время последнего сообщения** отображается справа
- **Правый клик на контакт** = удалить из контактов
- **Автоматическое добавление контактов** при первом сообщении

### 6. **Фишечки для удобства** 🎁
- **Синхронизация между вкладками одного браузера** через BroadcastChannel
- **Автосохранение** всех контактов и сообщений
- **Улучшенный поиск** с разделением на категории
- **Контекстное меню** для удаления контактов
- **Быстрое добавление контактов** при нахождении в поиске

## 🚀 Как использовать:

### Регистрация и вход:
1. Открыть приложение
2. **Регистрация**: ввести никнейм (мин 3 символа) и пароль (мин 4 символа)
3. **Вход**: ввести никнейм и пароль

### Добавление контакта:
1. Открыть **поле поиска** в левой панели
2. Начать вводить имя контакта
3. В разделе "🔍 Найти пользователей" нажать на контакт
4. Контакт добавится и откроется чат

### Синхронизация между браузерами:
1. **Браузер 1**: зарегистрировать "Маша", отправить сообщение "Привет"
2. **Браузер 2**: зарегистрировать "Говно"
3. В браузере 2 в поиске найти "Маша" и нажать
4. Там будет сообщение от Маши
5. Ответить "Привет, Маша!" 
6. Вернуться в браузер 1 - ответ будет виден

### Удаление контакта:
- **Правый клик** на контакт в списке
- Подтвердить удаление

## 📝 Технические детали:

### localStorage ключи:
- `zapretka_registered_users` - все зарегистрированные пользователи и пароли
- `zapretka_messages_[username]` - сообщения для конкретного пользователя
- `zapretka_contacts_[username]` - контакты конкретного пользователя
- `zapretka_user_[username]` - профиль пользователя

### Новые функции в data.js:
- `getUserContacts(username)` - получить контакты пользователя
- `saveUserContacts(username, contacts)` - сохранить контакты
- `addUserContact(username, contactName)` - добавить контакт
- `removeUserContact(username, contactName)` - удалить контакт
- `getAllRegisteredUsers()` - получить всех зарегистрированных пользователей
- `searchUser(query)` - поиск пользователя

### BroadcastChannel события:
- `contacts_update` - обновление контактов между вкладками
- `messages_update` - обновление сообщений между вкладками

## 🐛 Известные ограничения:

1. **Нет настоящей синхронизации между браузерами в режиме реального времени** - нужен активный WebSocket сервер
2. **Пароли хранятся в plain text** - в реальном приложении нужно использовать хеширование (bcrypt)
3. **Нет шифрования сообщений** - для приватности нужно добавить E2E шифрование
4. **Аватары генерируются через API** - при работе offline может быть проблема
